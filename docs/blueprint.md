# **App Name**: ClickSync

## Core Features:

- Google Sign-In: Allow users to sign in using their Google accounts with Firebase Authentication.
- Real-time Counter: Display a real-time counter that increments when a button is pressed, using Firebase.
- Click Event Listener: Implement a button that, when clicked, updates the counter value in the Firebase database. Uses Firebase Functions for scalability.
- Data Persistence: Persist the counter value in Firestore to maintain the count across sessions.

## Style Guidelines:

- Primary color: Soft blue (#75A9FF) to represent connectivity and calmness.
- Background color: Very light blue (#EAF2FF) to provide a clean and unobtrusive backdrop.
- Accent color: Warm orange (#FFB375) to draw attention to interactive elements.
- Body and headline font: 'Inter' sans-serif for a modern and readable interface.
- Use minimalist icons for actions and navigation.
- Clean, centered layout to focus user attention on the counter and button.
- Subtle button animations upon click to provide feedback.
